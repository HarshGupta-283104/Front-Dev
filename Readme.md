# Frontend Development Exercises

This repository contains 10 CSS and HTML exercises designed to help you understand and practice various frontend development concepts.

## Exercise List

1. **Pseudo-Classes Demonstration**
   - Location: `01-PseudoClasses/`
   - Demonstrates :hover, :focus, :active, and :nth-child() pseudo-classes
   - Shows state changes with colors and transitions

2. **Tooltip Demonstration**
   - Location: `02-Tooltips/`
   - Shows CSS-only tooltips with smooth transitions
   - Demonstrates positioning and opacity animations

3. **Image Showcase**
   - Location: `03-ImageShowcase/`
   - Features responsive grid layout
   - Shows hover effects with scaling and overlays

4. **CSS Selectors Practice**
   - Location: `04-Selectors/`
   - Demonstrates various selector types
   - Includes examples of combinators and attribute selectors

5. **Button Collection**
   - Location: `05-ButtonCollection/`
   - Shows different button styles and states
   - Includes hover and focus effects

6. **CSS Animations**
   - Location: `06-Animations/`
   - Demonstrates @keyframes animations
   - Shows various animation properties and timing functions

7. **UI Components**
   - Location: `07-UIComponents/`
   - Contains search bar, buttons, and dropdown menu
   - Uses consistent styling and interactions

8. **Box-Sizing Comparison**
   - Location: `08-BoxSizing/`
   - Compares content-box vs border-box
   - Visual demonstration of box model differences

9. **Combined Effects**
   - Location: `09-CombinedEffects/`
   - Shows multiple CSS effects working together
   - Demonstrates transitions and transformations

10. **Mini User Interface**
    - Location: `10-MiniUI/`
    - Complete small webpage with form elements
    - Shows practical application of learned concepts

## How to Use

1. Each exercise is in its own directory
2. Open the `index.html` file in your browser
3. View source code to understand implementation
4. Modify code to experiment with different values
5. Check comments for explanations

## Requirements

- Modern web browser
- Basic understanding of HTML and CSS
- Text editor for viewing and editing code

## Learning Goals

- Understanding CSS selectors and specificity
- Mastering layouts with Flexbox and Grid
- Creating smooth animations and transitions
- Building responsive designs
- Implementing modern UI components
- Writing semantic HTML

## Notes

- No JavaScript is used - all functionality is CSS-only
- Exercises use modern CSS features
- Code is well-commented for learning purposes
- All examples are responsive and mobile-friendly
